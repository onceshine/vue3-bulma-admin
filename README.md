# vue3-bulma-admin

vue3.0 试玩

使用 [vue-next](https://github.com/vuejs/vue-next) composition-api，同时涵盖 vue-router 以及 vuex

页面及样式来自 [Bulma](https://github.com/jgthms/bulma), [BulmaTemplates](https://github.com/BulmaTemplates/bulma-templates)

## Install

``` shell
npm install
```

## Usage

**Develop**

``` shell
npm run dev
```

访问 localhost:8080

**Build**

``` shell
npm run build
```
